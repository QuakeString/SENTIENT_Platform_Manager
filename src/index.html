System check

Making sure your PC is ready. Nothing is changed at this step.

Set up WSL2

SENTIENT runs in Docker inside WSL2 (Windows Subsystem for Linux). This installs and enables it.

This can take several minutes — the window stays responsive, please wait.


        


        

          
Restart required

          
Windows needs to restart to finish enabling WSL2. The installer will
            resume automatically after you sign back in.

          

            
            
          

        


        

          
✓ WSL2 is ready

          
WSL2 is installed and running.

        


        

          
          
        

      


      
      

        
Docker Engine

        
Creates a dedicated Ubuntu distro inside WSL2 and installs Docker Engine in it.


        

          
        


        

          

          

          
Downloads ~350 MB and installs Docker — a few minutes. The window stays responsive.

          

        


        

          
✓ Docker Engine is ready

          
The dedicated SENTIENT distro is set up with Docker running.

        


        

          
          
        

      


      
      

        
Deploy SENTIENT

        
Pull the SENTIENT images and start the stack. It'll be at localhost:8080 when ready.


        

          
        


        

          

          

          
First run pulls ~1 GB of images and initializes the database — a few minutes. The window stays responsive.

          

        


        

          
✓ SENTIENT is running

          
Open http://localhost:8080 and sign in as sys-admin:

              admin@sentient.local  /  admin123

            It starts automatically on login from now on.
